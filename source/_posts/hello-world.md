# Hexo 语法入门

## Create a new post

``` bash
$ hexo new "My New Post"
$ hexo g
$ hexo s
$ hexo d
```
<!--more-->

## Generate a static file

``` bash
$ hexo generate
```


## Run on server

``` bash
$ hexo server
```


## Deploy to remote sites

``` bash
$ hexo deploy
```
