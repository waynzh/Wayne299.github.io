VSCode 好用的快捷键

FE
  • tools
  • vscode

光标移动

整体移动一行

move line up

  • ⌥ + 上/下

下一个单词

  • ⌥ + 左/右

删除一个单词

  • ⌥ + Delete

删除一行

delete line

  • 删除：⌘ + ⇧ + K
  • 直接剪切掉：⌘ + x

批量操作

多行合并为一行

join lines

  • ⌃ + J

批量选择

  • ⌘ + D

批量复制光标以添加内容

add cursor below

  • ⌘ + ⌥ + 上/下

批量替换当前文件中所有匹配的文本

change all

  • ⌘ + F2

激活/聚焦

打开最近使用的文件夹

open recent

  • ⌃ + R
  • 配合 window.nativeTabs，全屏下自动在New Tab中打开，非全屏时会打开New Window

全局查找时逐项查看

  • F4

下一个Terminal / 下一个编辑器 Tab页

open previous editor / focus previous terminal group

  • ⌥ + ⌘ + 左/右


同一 Terminal Group中切换

focus previous terminal in terminal group

  • ⌥ + ⌘ + 上

focus 不同的编辑分区

focus editor group

  • ⌘ + 数字键

移动 编辑器到其他分区

move editor

  • ⌃ + ⌘ + 左/右
